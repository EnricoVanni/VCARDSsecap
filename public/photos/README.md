# Foto profilo opzionali

Posiziona qui le foto profilo dei dipendenti, nominate come <slug>.jpg
(es. m.rossi.jpg per il dipendente con slug "m.rossi").

Consigli:
- Formato JPEG, quadrate (1:1)
- Risoluzione: 400x400 px (sufficiente per il medaglione)
- Peso: max 50KB ognuna (vengono embeddate in base64 nella vCard)

Se una foto manca, la pagina mostra automaticamente le iniziali come monogram.
